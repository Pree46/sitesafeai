# FFMPEG SETUP GUIDE

This project uses FFMPEG to run media on browser.

---

## 1. Setup

- Go to https://ffmpeg.org/download.html
- Click on Windows icon.
- You'll see options. Click on "build by BtbN". (You will redirected to a github page)
- Download ffmpeg-n8.0-latest-win64-gpl-8.0.zip. (or the latest version at the time of download)
- Extract the folder to C drive
- go to the bin folder and copy the path
  Example: C:\ffmpeg-n8.0-latest-win64-gpl-8.0\bin
- Add it to your Windows PATH in system Variables.
- Restart your terminal
